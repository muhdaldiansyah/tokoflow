## Why Alexa won homes / Siri lost pockets

Sources: [therobowire 2026 comparison](https://therobowire.com/voice-assistant-comparison-alexa-google-siri-2026/), [gearbrain — Apple Siri left behind](https://www.gearbrain.com/apple-siri-ai-alexa-google-2499002583.html), [Quora — Why Alexa more successful](https://www.quora.com/Why-has-Alexa-been-more-successful-than-Siri).

### 5 lessons mapped to Tokoflow

1. **Open ecosystem beats walled garden.** Alexa: 140K+ skills, 100K+ devices via simple software update. HomeKit: hardware certification required. → Tokoflow: accepts any QRIS PJP, any WA contact, any merchant-existing-stack. Not walled. ✓ already in current-best.

2. **The "novelty trap" kills voice products.** Siri felt like a gimmick after the first few demos because it lacked a daily-use anchor. → Tokoflow's anchors: morning briefing (Muslim Pro analog) + WA receipt loop closure + diary-IS-DB queries. Voice gets used because it's COMMERCIALLY anchored, not novelty-anchored. ✓ already in current-best.

3. **Don't position as "voice assistant."** Siri is the cautionary tale. Voice-as-feature is fragile. → Tokoflow positions as voice-DIARY (Cerita je / Cerita aja) — voice is the input mode for a specific commercial diary, not a general assistant trying to do everything. ✓ already in current-best.

4. **No wake word for Tokoflow — tap-to-talk explicit.** Wake words add creep factor (Sari sev-8 privacy concern + Aisyah "kids tertekan tetiba transcript ada cerita rahsia"). Lock-screen widget = intentional, biometric-gated activation. → ✓ already in current-best (cycle 5 lateral refinement).

5. **Context > capability.** Alexa won home because home is the right context for "play music, dim lights". Siri lost pocket because pocket is the wrong context for general AI (privacy, situational appropriateness, typing wins for some tasks). → Tokoflow wins kedai-context because kedai-context is right for "Aishah dah bayar RM 25" (hands-busy, scattered, social-acceptable to talk to phone in working environment, not in public meeting). Don't try to be a general voice assistant.
