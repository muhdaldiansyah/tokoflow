# Cycle 022 — Bu Aisyah critique (UX-C, mental simulation)

Shah Alam, 80 orders/bulan, 3 budak, hari berserabut. Voice diary user since launch.

## 1. Now section (pinned top)
**YA, ni yang aku nak.** Pagi tadi tengah uli dough, anak tengah nangis, aku just nak tau "siapa belum bayar, siapa pickup hari ni". Now section bagi aku 3-saat glance — tak payah scroll, tak payah filter. Bukan nav layer, sebab dia derived dari diary aku sendiri (bukan CRM tab). **Match real need.**

## 2. Mic FAB (bottom)
Tangan aku selalu berlumur tepung/minyak. Large breathing button = tap senang. **TAPI** — honestly aku jarang buka app. 80% aku guna lock-screen widget time pickup masuk. FAB perlu untuk reconcile petang, but lock-widget = primary. Both kena ada. ✓ kalau kedua-dua sama smooth.

## 3. Section collapsing (Yesterday ▶ vs Today ▼)
**Match.** Aku tak nak scroll panjang sampai semalam — kalau perlu, aku tap. Default Today expanded = betul. Cuma satu concern — **"This week" / "This month" rasa macam folder, bukan diary**. Boleh ke just "Today" + "Earlier"? Dua je, lagi simple. Severity 5.

## 4. Tap Now card → scroll to source voice note
**Aku faham, tapi first time confusing.** My instinct: tap pending payment → claim button terus. Scroll ke voice note rasa extra step. Diary mental model preserved YES, tapi kena ada inline action button kat Now card jugak (Claim / Mark paid). Kalau nak detail baru scroll. **Severity 7** — kalau setiap tap = scroll, lunch rush aku marah.

## 5. Lunch rush mode
Helpful, **kalau pulse subtle betul-betul**. Kalau FAB pulse macam notification merah — aku panic lagi. Reorder Now (pending payment dulu) = berguna. ✓

## 6. Quiet hours — tomorrow-prep panel at 10pm
**Severity 7 concern.** 10pm aku dah baring dengan budak. Buka app sebab nak check satu benda — terus nampak "Pak Lee Sabtu" — rasa macam kerja tarik aku balik. Lagi baik: app open biasa, tomorrow-prep ada **kalau aku scroll**, jangan auto-pin. Opt-in, jangan push.

## 7. No streaks/badges/comparison
**Lega.** Aku dah cukup pressure dari mak mertua. Tokoflow jangan jadi mak mertua kedua. ✓✓✓

## ≥7 severity concerns
- **#4**: Now card tap = scroll only. Need inline Claim/Paid action. Otherwise lunch rush = friction.
- **#6**: Quiet-hours auto-pin tomorrow-prep = work-after-work. Make it scroll-discoverable, not pinned.

## Scores
- UX matches my real day: **8/10**
- Greasy fingers + 3 kids: **8/10** (lock-widget kena solid)
- Diary mental model preserved: **9/10**
- Want to use daily: **8/10** (fix #4 dan #6, jadi 9)
