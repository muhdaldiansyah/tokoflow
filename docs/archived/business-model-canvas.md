# Business Model Canvas — CatatOrder

> Link Toko & Kelola Pesanan untuk UMKM Indonesia
> catatorder.id | v2.4.0 | Maret 2026

---

## Ringkasan

CatatOrder adalah alat link toko dan pengelolaan pesanan untuk UMKM Indonesia yang terima pesanan lewat WhatsApp. Pelanggan pesan lewat link toko, pesanan langsung tercatat di dashboard. 50 pesanan/bulan gratis dengan semua fitur — bayar mulai Rp15 ribu kalau butuh lebih.

## 2. Value Propositions

### Masalah Inti

85% UMKM F&B kelola pesanan manual lewat WhatsApp — **77% UMKM masih mencatat keuangan secara manual** (OCBC BFI 2024 x NielsenIQ). 15-30% pesanan hilang atau tertukar. 1-3 jam/hari habis untuk catat dan rekap. 98% tidak punya pembukuan sama sekali.

### 9 Masalah yang Diselesaikan

| # | Masalah | Tingkat | Solusi CatatOrder |
|---|---------|---------|------------------|
| 1 | Pesanan tercecer di chat WA | Kritis | Link toko + dashboard terstruktur |
| 2 | Catat manual rawan salah | Kritis | Kalkulasi otomatis, AI parsing |
| 3 | Utang pelanggan tidak terlacak | Kritis | Lacak pembayaran (DP/Lunas) |
| 4 | Struk tulis tangan tidak profesional | Tinggi | Struk digital via WA |
| 5 | Tidak ada status pesanan | Tinggi | Alur: Baru → Diproses → Dikirim → Selesai |
| 6 | Rekap akhir hari 30-60 menit | Tinggi | Rekap otomatis + laporan bulanan |
| 7 | Alat yang ada terlalu rumit | Sedang | UI ultra-sederhana, onboarding < 2 menit |
| 8 | Alat yang ada terlalu mahal | Sedang | Gratis 50 pesanan, Rp15K isi ulang |
| 9 | Tidak ada alat untuk SETELAH pesanan masuk | Tinggi | Satu-satunya tool order-first (bukan catalog-first) |

### Proposisi Unik

**Satu-satunya tool order-first di pasar.** 12 tool WA commerce global di ProductHunt semuanya catalog-first (bangun toko, tampilkan produk). Tidak ada yang mengelola siklus hidup pesanan setelah masuk. CatatOrder adalah satu-satunya yang fokus pada: pesanan masuk → status → pembayaran → struk → rekap.

**Harga Rp15K tanpa kompetitor.** Gap antara aplikasi gratis (BukuWarung) dan POS Rp55K+ (Kasir Pintar). CatatOrder sendirian di Rp15K. Selly (1 juta+ transaksi/bulan) tutup Juli 2025 — digantikan QuickOrder.ai (keyboard app, bukan order management). Dazo (dazo.id) = kompetitor terdekat (AI + WA Bot), tapi berbeda positioning.

### Dampak Finansial untuk User

| Masalah | Kerugian | Dasar |
|---------|----------|-------|
| Pesanan hilang/salah | Rp500K+ per kejadian | Rata-rata order kue |
| Waktu rekap manual | Rp10-20K/hari | 30-60 menit x Rp20K/jam |
| Utang tidak tercatat | Rp500K-5 juta terkunci | Riset hutang warung |
| Waktu terhemat dengan otomasi | 50-60% | Studi kasus Kitalulus |

## 3. Channels

### Distribusi (urut prioritas)

**1. WA Viral Loop — Rp0, potensi tertinggi**

Setiap pesan WA yang dikirim dari CatatOrder (konfirmasi, struk, pengingat) mengandung branding: *"Dibuat dengan CatatOrder — catatorder.id"*. Link toko (`catatorder.id/[slug]`) terlihat setiap kali pelanggan pesan.

Preseden: BukuWarung tumbuh ke 7 juta user, 60-70% dari WA viral loop saja.

Hitungan: 100 user aktif x 10 WA branded/bulan x 1% klik x 50% daftar = 5 user baru/bulan (compound).

**2. TikTok / Reels — Rp0, pertumbuhan awal tercepat**

125-157 juta user TikTok di Indonesia. Konten UMKM performa baik. Target: 3-5 video/minggu. Estimasi: 20-30 signup/bulan dari 20 video/bulan.

**3. Pain Point SEO — Rp0, aset jangka panjang**

<1% konten internet dalam Bahasa Indonesia = kompetisi sangat rendah. Target keyword: "pesanan WA berantakan", "aplikasi catat orderan". Estimasi: 75-150 signup/bulan dari SEO di bulan 12-18.

**4. WA Group Seeding — Rp0, kickstart manual**

Komunitas besar: UKMINDONESIA.ID (40K+), Tangan di Atas (42K+), HaiBolu (75K+), Yuk Bisnis (75K+). Target: 2-7 grup untuk 50-100 user pertama.

### Bagaimana UMKM Menemukan Alat (urut kepercayaan)

1. Rekomendasi teman / mulut ke mulut (80%+ adopsi)
2. Grup WhatsApp (sharing komunitas)
3. Media sosial (Instagram, TikTok)
4. Google Search (Bahasa Indonesia)
5. Program pemerintah

### Biaya Distribusi: Rp0

BukuKas habiskan $80 juta untuk iklan → 6,3 juta user, 95% tidak aktif → bangkrut. CatatOrder: WA viral (Rp0) + TikTok (Rp0) + SEO (Rp0) + outreach manual (Rp0).

## 4. Customer Relationships

### Self-Serve (Utama)

- Daftar 1 tap via Google OAuth, onboarding < 2 menit
- Free tier 50 pesanan/bulan — tanpa komitmen
- 100% Bahasa Indonesia, touch target besar (44px min)
- Aturan "ribet" 3 langkah dihormati (>3 interaksi = ditinggalkan)

### Personal (Fase Awal, 0-100 user)

- Onboarding manual via WA video call untuk 5-10 user pertama
- Follow-up WA di Hari 0/1/3/7/14/30
- Support personal via WA (98% open rate vs 20% email)

### Otomatis (100+ user)

- Drip sequence WA onboarding otomatis
- Nudge banner progresif di pesanan ke-40/45/48/50
- Notifikasi realtime + suara untuk pesanan masuk
- Laporan nilai bulanan

### Mekanisme Retensi

| Mekanisme | Cara Kerja |
|-----------|-----------|
| IKEA Effect | User bangun database pelanggan + riwayat pesanan = nilai tinggi yang dipersepsikan |
| Endowment Effect | Data gratis yang dimasukkan menciptakan rasa kepemilikan |
| Status Quo Bias | Begitu CatatOrder jadi default, biaya pindah sangat besar |
| "Menunggu" | Pesanan pelanggan tetap masuk tapi menunggu → urgensi bayar, bukan pilihan |
| Habit Loop | Target: 66 hari penggunaan harian = kebiasaan permanen |

### Benchmark Aktivasi

| Metrik | Target | Red Flag |
|--------|--------|----------|
| Aktivasi Day-1 | >40% | <20% |
| Retensi Day-7 | >25% | <10% |
| Retensi Day-30 | >15% | <5% |
| Churn bulanan (bayar) | <5% | >10% |
| Konversi gratis → bayar | 3-5% | <1% |

## 5. Revenue Streams

### Model Harga

| Paket | Harga | Detail |
|-------|-------|--------|
| **Gratis** | Rp0 | 50 pesanan/bulan. Semua fitur termasuk (link toko, struk, AI, rekap). Selamanya. |
| **Isi Ulang** | Rp15.000 | +50 pesanan. Tidak kadaluarsa. Bayar QRIS. |
| **Unlimited** | Rp35.000/bulan | Pesanan tak terbatas selama 1 bulan. Bayar QRIS. |

### Unit Economics (per Isi Ulang pack)

```
Pendapatan:          Rp15.000
Biaya QRIS (0,7%):   -Rp105
Infra (50 pesanan):   -Rp2.200
AI subsidi:           -Rp200
───────────────────────────────
BERSIH:              Rp12.495 (margin 83%)
```

### Proyeksi Revenue

| User Aktif | Gratis | 1 Pack | 2 Pack | Unlimited | MRR | Profit |
|------------|--------|--------|--------|-----------|-----|--------|
| **100** | 30 | 50 (Rp750K) | 15 (Rp450K) | 5 (Rp225K) | **Rp1,4 juta** | Rp1,3 juta |
| **500** | 150 | 250 (Rp3,75M) | 75 (Rp2,25M) | 25 (Rp1,125M) | **Rp8,25 juta** | Rp7,35 juta |
| **1.000** | 300 | 500 (Rp7,5M) | 150 (Rp4,5M) | 50 (Rp2,25M) | **Rp18 juta** | Rp16,8 juta |

### Break-Even

```
Fase 1 (Supabase Free + Vercel Hobby):
  Biaya: ~Rp62.500/bulan
  Break-even: 5 user beli 1 pack (5 x Rp15K = Rp75K)

Fase 2 (Supabase Pro + Vercel Pro):
  Biaya: ~Rp1.100.000/bulan
  Break-even: ~55 user beli 1 pack
```

### Kenapa Model Ini

| Pilihan | Alasan |
|---------|--------|
| Gratis 50/bulan | Persyaratan psikologis — UMKM tidak mau coba kalau bayar dulu. Viral loop butuh user aktif gratis. |
| Pack (bukan langganan) | Sachet economy: Rp500/pesanan mendapat 70-80% penerimaan vs 15-20% untuk Rp100K/bulan. |
| QRIS only | VA fee Rp4.000 = 26,7% dari Rp15K (prohibitif). QRIS fee Rp105 = 0,7% (negligible). |
| Rp15K | Magic price point Indonesia. Setara harga GoRide, 5 bungkus Indomie. Di bawah ambang Rp50K (tidak perlu diskusi pasangan). |

### Perbandingan Harga Pasar

```
Gratis (BukuWarung, pivot)  →  Rp15K (CatatOrder, SENDIRIAN DI SINI)  →  Rp55K+ (Kasir Pintar)  →  Rp249K+ (Majoo/Moka)
```

## Competitive Moat

| Moat | Mekanisme |
|------|-----------|
| **WA Viral Loop** | Setiap struk/konfirmasi = branding gratis + link balik (Rp0 CAC) |
| **Order-First** | Satu-satunya tool global yang fokus mengelola pesanan SETELAH masuk via WA |
| **Harga Kosong** | Rp15K — gap antara gratis (BukuWarung) dan Rp55K+ (Kasir Pintar). Selly (1M+ tx/bulan) tutup, Dazo satu-satunya kompetitor dekat |
| **Kesederhanaan** | Satu hal yang dilakukan dengan baik vs pendekatan "semua fitur" kompetitor |
| **Switching Cost** | Data pelanggan + riwayat pesanan yang terakumulasi setiap hari (IKEA Effect) |
| **Network Effect** | Lebih banyak penjual = lebih banyak branded receipt beredar = lebih banyak penjual baru |
| **Bahasa** | 100% Bahasa Indonesia informal — tool global semuanya English-first |
| **Horizontal Entry** | Tangkap semua vertikal UMKM, data penggunaan reveal vertikal untuk cross-sell |
