# Video 5 — Cerita User Nyata

> Testimonial dari UMKM yang benar-benar pakai CatatOrder. Paling powerful — tapi butuh user nyata dulu.

**Format:** Testimonial / Storytelling
**Durasi:** 45 detik
**Prioritas:** Setelah ada 1+ user aktif yang bersedia sharing

---

## Kenapa Video Ini

- Social proof = sinyal kepercayaan paling kuat untuk UMKM
- "Orang seperti saya sudah pakai" > "fitur ini canggih"
- Satu testimonial asli lebih powerful dari 10 video demo
- Tapi HANYA bisa dibuat setelah ada user nyata yang puas

---

## Script Template

### HOOK (0-3 detik)

**Visual:** Wajah user (selfie / video call)
**Text on screen:** "Dulu pesanan saya berantakan."
**Voiceover (user):** "Dulu saya catat pesanan di buku tulis..."

### PROBLEM (3-12 detik)

**Visual:** User cerita (talking head) + B-roll toko/dapur mereka
**Voiceover (user):** "Saya punya [usaha]. Setiap hari [jumlah] pesanan masuk dari WA. Saya catat manual, sering kelewat, pelanggan complain..."
**Text on screen:** Highlight angka: "20 pesanan/hari" "3x kelewat bulan lalu"

### SOLUTION (12-30 detik)

**Visual:** User tunjukkan CatatOrder di HP mereka (screen recording actual data mereka)
**Voiceover (user):** "Sekarang pakai CatatOrder — pelanggan pesan sendiri lewat link. Saya bangun pagi, pesanan sudah rapi. Tau siapa yang sudah bayar, siapa yang belum."
**Text on screen:** Highlight: "Link toko → pesanan otomatis" "Rekap harian"

### RESULT (30-40 detik)

**Visual:** User tersenyum + tunjukkan rekap/omzet
**Voiceover (user):** "Bulan ini [jumlah] pesanan tercatat. Omzet [angka]. Gak ada yang kelewat lagi."
**Text on screen:** Angka besar: "Rp X.XXX.XXX bulan ini"

### CTA (40-45 detik)

**Visual:** Layar CatatOrder / user bilang langsung
**Voiceover (user):** "Cobain aja, gratis. catatorder.id."
**Text on screen:** "Gratis. catatorder.id"

---

## Cara Dapat Testimonial

### Tahap 1: Minta langsung ke user aktif

Setelah ada 1 user yang aktif (5+ pesanan):
```
"Halo [nama]! Makasih sudah pakai CatatOrder.
Boleh minta tolong cerita pengalaman kamu?
Cuma 1 menit — bisa video selfie atau voice note.

Pertanyaan:
1. Usaha apa dan biasanya terima berapa pesanan/hari?
2. Sebelum CatatOrder, gimana caranya catat pesanan?
3. Sekarang pakai CatatOrder, apa yang berubah?
4. Ada angka yang bisa di-share? (pesanan/bulan, omzet)

Kalau mau video selfie lebih bagus, tapi voice note juga ok!"
```

### Tahap 2: Video call + screen recording

Jika user bersedia:
1. Video call via WA — minta mereka cerita
2. Minta mereka screen-share CatatOrder di HP mereka
3. Record semua — edit jadi 45 detik

### Tahap 3: User-generated

Jika sudah punya 10+ user aktif:
- Buat template: "Ceritakan pengalaman pakai CatatOrder — tag @catat.order"
- Repost/duet video user di TikTok

---

## Jangan Dipaksakan

Testimonial palsu / script yang dibuat-buat terasa LANGSUNG di TikTok. UMKM audience sangat sensitif terhadap yang tidak autentik.

Lebih baik tidak ada testimonial daripada testimonial yang terasa dipaksakan.
