# Video CatatOrder

> Rencana produksi video demo — dari script sampai distribusi.

---

## Strategi

**Tujuan:** Video adalah Encounter (QDA Step 0) — membuat calon user bertanya "ada cara lebih baik?" dan langsung signup.

**Platform target:**
- TikTok (125M+ user Indonesia) — primary
- Instagram Reels — repurpose
- WA Status — clip pendek
- Facebook Groups UMKM — share langsung

**Gaya produksi:**
- Screen recording HP asli (bukan mockup/animasi)
- Voiceover santai bahasa sehari-hari
- Vertical 9:16
- Authentic > polished (UMKM skip video yang terasa iklan)

**Flow konversi:** Lihat video → "ini masalah saya" → klik link → signup → pesanan pertama → kirim WA

## Distribusi per Video

```
1 video TikTok
  → repost ke Instagram Reels (same file)
  → crop 15-30 detik untuk WA Status
  → screenshot momen key → IG carousel
  → transcribe insight → post di FB Group UMKM
```

1 produksi = 4-5 touchpoint distribusi.

---

## Aturan Produksi

1. **Hook 3 detik pertama** — 63% CTR tinggi karena hook di 3 detik. Kalau kehilangan viewer di sini, semuanya sia-sia.
2. **Tunjukkan, jangan jelaskan** — demo > penjelasan. Magic moment harus TERLIHAT, bukan diceritakan.
3. **Satu video = satu pesan** — jangan coba cover semua fitur dalam 1 video.
4. **CTA sederhana** — "Gratis. catatorder.id" atau "Link di bio." Tidak lebih.
5. **Iterasi** — video pertama 10K views? Buat 3 variasi. < 100 views? Ganti hook, coba lagi.

## Tools

- **Record:** built-in screen recorder iOS/Android
- **Edit:** CapCut (gratis, support text overlay + voiceover)
- **Post:** TikTok app → auto-repost ke IG Reels
- **Track:** TikTok Analytics (views, shares, saves)
