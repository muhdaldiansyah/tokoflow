## Direct Competitor #1: Manual WhatsApp

**The actual incumbent.** Most UMKM manage orders by scrolling WhatsApp chat history.

| Attribute | Value |
|-----------|-------|
| Users | Millions of UMKM |
| Price | Free |
| Method | Type orders in chat, screenshot for record, manual recap |

**Pain points users report:**
- "Orders scattered across chats" — can't find old orders
- "Miss orders, forget to ship" — no status tracking
- "Manual recap to spreadsheet" — time-consuming end-of-day
- "Capek ngetik invoice berulang kali" — fatigue from repetitive typing
- "Kewalahan membalas satu per satu" — overwhelmed replying one by one

**CatatOrder advantage:** Structured order entry, auto-recap, digital receipts — all without leaving WA workflow.

## Direct Competitor #2: AkunIndo

| Attribute | AkunIndo | CatatOrder |
|-----------|----------|------------|
| Status | Beta (free) | LIVE (pricing active) |
| WhatsApp input | Share TO app, not FROM | WhatsApp-native workflow |
| Web dashboard | Coming V3 | Available now |
| Tax reports | No | No (not needed for order mgmt) |
| Receipt/struk | Basic | Photo OCR + digital receipts |
| Target | Receipt/bookkeeping | Order management |
| Pricing | Unproven | Gratis/Rp49K/Rp99K |

**Gap:** AkunIndo is focused on receipt scanning for bookkeeping. CatatOrder is focused on order management and digital receipts for WA-based businesses. Different use case.

## Direct Competitor #3: BukuWarung

| Attribute | Value |
|-----------|-------|
| Users | 7M+ claimed (declining: ~20K downloads/month) |
| Funding | $100M+ (Valar, Y Combinator, DST Global) |
| Price | Free |
| Primary function | Bookkeeping / ledger |
| Stock management | Basic — just recording, no alerts or analytics |
| Order management | None |
| Revenue model | Commission from PPOB, digital payments, lending |

**Current status (2025-2026):**
- Revenue grew 72% to $1.7M in 2023 (still tiny vs $100M funding)
- Slashed losses by 75% through cost cutting
- **Pivoting from bookkeeping toward fintech/payments**
- Play Store reviews: "seems to be downgrading" / "feature bloat"

**Gap for CatatOrder:** BukuWarung is abandoning its core warung bookkeeping to chase fintech revenue. CatatOrder captures the order management need BukuWarung never addressed. BukuWarung = ledger. CatatOrder = orders.

## Direct Competitor #4: WA Business Tools (Wati, Qontak, Interakt)

| Tool | Pricing | Focus | Gap |
|------|---------|-------|-----|
| Wati | $49+/month | WA API + automation | English-first, enterprise pricing |
| Qontak (Mekari) | Enterprise pricing | Omnichannel CRM | Way too expensive for UMKM |
| Interakt | $19+/month | WA CRM | Not order management |
| EVA | Custom pricing | Chatbot | Not order management |

**Gap:** No Rp49K WhatsApp order management for UMKM. These tools are chatbots and CRM, not order tracking.
