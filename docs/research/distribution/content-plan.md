# Content Plan — CatatOrder

> TikTok/Reels content formats + frequency for $0 distribution.

---

## Why TikTok for UMKM

- 125M Indonesian TikTok users (DataReportal 2024)
- Algorithm-driven: followers don't matter initially
- Indonesian UMKM owners are on TikTok (both as consumers and creators)
- Short demo videos of "solving a real problem" perform well
- Zero cost: phone screen recording + CapCut editing
- Comments section = free market research
- None of the 5 audited SaaS companies used TikTok (opportunity)

---

## Content Formats

### Format 1: Problem → Demo (Primary)

| Element | Content |
|---------|---------|
| Hook (3 sec) | "Masih catat order di WA yang bertumpuk?" |
| Body (20-40 sec) | Screen recording: open CatatOrder → add order → generate receipt → send WA |
| CTA (3 sec) | "Link di bio — gratis!" |
| Duration | 30-60 seconds |

### Format 2: Before → After

| Element | Content |
|---------|---------|
| Hook | "Dulu: buku + kalkulator + scroll WA" |
| Body | Split screen: Left = WA chaos / Right = CatatOrder dashboard |
| CTA | "Coba gratis: link di bio" |
| Duration | 15-30 seconds |

### Format 3: Relatable Pain (Storytelling)

| Element | Content |
|---------|---------|
| Hook | "1 order hilang = Rp500K melayang" |
| Body | Story: baker lost order in WA chat. Rp500K gone. "Tahun ini dia pakai ini..." → show dashboard |
| CTA | "Jangan sampai keulang. Link di bio." |
| Duration | 30-45 seconds |

### Format 4: Tutorial (Educational)

| Element | Content |
|---------|---------|
| Hook | "Cara bikin struk digital tanpa printer" |
| Body | Step-by-step: buat pesanan → generate struk → kirim ke WA customer |
| CTA | "Gak perlu modal printer thermal. Link di bio." |
| Duration | 30-45 seconds |

### Format 5: Behind the Scenes (Trust)

| Element | Content |
|---------|---------|
| Hook | "Saya developer yang bikin app untuk UMKM..." |
| Body | Personal story, show building process, show real users |
| CTA | "Kalau punya bisnis yang terima order via WA, coba ini" |
| Duration | 30-60 seconds |

### Format 6: Testimonial (Social Proof)

| Element | Content |
|---------|---------|
| Hook | "Ibu [nama] pemilik [bisnis] di [kota]" |
| Body | User cerita pengalaman (WA voice note + dashboard screenshots) |
| CTA | "Mau seperti Ibu [nama]? Link di bio." |
| Duration | 30-60 seconds |

### Format 7: Tips/Education (Value-First)

| Element | Content |
|---------|---------|
| Hook | "3 Cara UMKM Nggak Kehilangan Order Lebaran" |
| Body | Educational tips, mention CatatOrder at tip #3 |
| CTA | "Tip terakhir: pakai CatatOrder. Gratis." |
| Duration | 60 seconds |

## Repurpose Strategy

Every TikTok video gets repurposed:

```
1 TikTok video → Instagram Reels (same video)
                → IG Carousel (screenshots from video)
                → FB Group post (transcribe key insight)
                → WA Status (15-sec clip)
```

**1 production effort = 4-5 distribution touchpoints.**

## Frequency & Schedule

| Phase | Frequency | Total/Month |
|-------|-----------|-------------|
| Week 1-2 (Seeding) | 1 video every 2 days | 7-8 videos |
| Week 3-4 (Ramadan) | 1 video every 2 days | 7-8 videos |
| Ongoing (Post-Lebaran) | 3-5 videos/week | 15-20 videos |

### Best Posting Times (Indonesian UMKM)
- Morning: 7-9 AM (before they start working)
- Evening: 7-9 PM (after work, scrolling time)
- During Ramadan: After sahur (4-5 AM) and after buka puasa (6-7 PM)

## Content Metrics to Track

| Metric | What It Tells You |
|--------|-------------------|
| Views | Reach (algorithm performance) |
| Completion rate | Content quality (hook + body) |
| Profile visits | Interest level |
| Link clicks | Conversion intent |
| Comments | Engagement + market research |
| Saves | High-value content signal |

### Iteration Rules
- Video gets 10K+ views? Make 3 variations of the same format.
- Video gets <100 views? Try different hook, same content.
- Comments asking same question? Make a video answering it.
- DM everyone who comments or likes — they're already interested.
