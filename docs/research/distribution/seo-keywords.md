## Content Templates (SEO Articles)

### Template 1: Pain-Focused Article
```
Title: "X Masalah [Bisnis Owner] dan Solusinya"
Structure:
  1. Hook: masalah yang reader rasakan (2-3 kalimat)
  2. List of pains with real examples
  3. Manual solutions (acknowledge their current method)
  4. Better solution: introduce CatatOrder
  5. Demo / screenshot / before-after
  6. CTA: "Coba gratis di catatorder.id"
Target: 1,000-2,000 kata
```

### Template 2: Tutorial Article
```
Title: "Cara Mengelola [Proses] dengan Mudah"
Structure:
  1. Problem statement
  2. Step-by-step manual method
  3. Step-by-step with CatatOrder (easier)
  4. Before/after comparison
  5. CTA: sign up gratis
```

### Template 3: Comparison Page
```
Title: "CatatOrder vs Catat Manual di WA"
Structure:
  1. Side-by-side comparison table
  2. Screenshots of both methods
  3. Time savings calculation
  4. Free trial CTA
```

### Template 4: Lead Magnet
```
Title: "Template Nota/Struk [Bisnis Type] Gratis"
Structure:
  1. Free downloadable template
  2. How to use it
  3. Even easier: use CatatOrder for auto-generation
  4. CTA: sign up free
```
